## Помощь по использованию
![help](img/help.png)

1. **``Title`` - название окна | заголовок**
2. **``Message`` - текст в уведомлении**
3. **``app_name`` - название приложения**
4. **``timeout`` - сколько времени будет отображаться уведомление (только целые числа)**
